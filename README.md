# 🔐 Password Security Toolkit

A comprehensive password security suite with **Password Manager**, **Strength Analyzer**, and **Password Generator** - all in one CLI tool.

## 👨‍💻 Author
**Gaurav Yadav**

## ✨ Features

### 🔐 Password Manager
- AES-256 like encryption for secure storage
- Master password protected vault
- Add, retrieve, update, and delete passwords
- Search functionality
- Password strength report for all entries
- Local storage only - your data stays with you

### 🔍 Password Analyzer
- Password strength scoring (0-100)
- Entropy calculation (bits of randomness)
- Crack time estimation
- Character variety analysis
- Common password detection
- Pattern recognition (keyboard sequences, repetition)

### 🔐 Password Generator
- Generate strong random passwords
- Customizable length (12-32 characters)
- Includes uppercase, lowercase, numbers, and special characters
- Multiple password generation at once
- Entropy display for each password
